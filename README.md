# al-hasanah

Modern e-commerce storefront built with Vue 3 and Vite. We start in JavaScript and migrate to TypeScript later.

## Requirements
- Node.js (current LTS recommended)
- pnpm

## Install
```sh
pnpm install
```

## Run in Development
```sh
pnpm dev
```

## Build for Production
```sh
pnpm build
```

## Preview Production Build
```sh
pnpm preview
```

## Scripts
- `pnpm dev` start the dev server
- `pnpm build` build for production
- `pnpm preview` preview the production build locally

## Goals
- Clean, teachable structure
- Best practices and modern tooling
- Small, composable components
