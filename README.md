# al-hasanah

Modern e-commerce storefront built with Vue 3 + Vite + Pinia + GraphQL.
We start in JavaScript and migrate to TypeScript later.

## Docs
- `docs/steps-100.md` — 100-step learning + build plan
- `docs/command-log.md` — every command we run and why
- `docs/api.md` — API reference and GraphQL notes

## Stack (target)
- Vue 3 (latest stable at install time)
- Vite (latest stable at install time)
- Pinia (latest stable at install time)
- Tailwind CSS (latest stable at install time)
- Apollo Client (latest stable at install time)

## Project Goals
- Clean, teachable structure
- Best practices and modern tooling
- Small, composable components
- Clear documentation
